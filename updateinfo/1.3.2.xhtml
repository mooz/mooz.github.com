KeySnail - Changelog

Version 1.3.2

  • Added popup help of the local keymap in prompt.selector().
    • If you forget the which key you bound the action, just press C-? or click the question-mark icon on the prompt.
    • prompt.selector() is the interface which adds anything.el feelings to Mozilla!
    • Most plugins like Yet Another Twitter Client KeySnail, Tanything, Keydledo and so forth using this interfaces.
    • When no local keymap is specified in that session, no popup will be displayed.
